Perform the following temperature unit conversions:

\begin{itemize}
\item{} 540$^{o}$ C = \underbar{\hskip 50pt} $^{o}$ F
\vskip 5pt
\item{} 1235 K = \underbar{\hskip 50pt} $^{o}$ C
\vskip 5pt
\item{} 80$^{o}$ F = \underbar{\hskip 50pt} $^{o}$ R
\end{itemize}

\underbar{file i00638}
%(END_QUESTION)





%(BEGIN_ANSWER)

\begin{itemize}
\item{} 540$^{o}$ C = \underbar{\bf 1004}$^{o}$ F
\vskip 5pt
\item{} 1235 K = \underbar{\bf 961.85}$^{o}$ C
\vskip 5pt
\item{} 80$^{o}$ F = \underbar{\bf 539.67}$^{o}$ R
\end{itemize}

2 points for C-F conversion, 4 points each for K-C and F-R conversions.
