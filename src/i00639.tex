Perform the following temperature unit conversions:

\begin{itemize}
\item{} 450$^{o}$ F = \underbar{\hskip 50pt} $^{o}$ C
\vskip 5pt
\item{} 2135 K = \underbar{\hskip 50pt} $^{o}$ C
\vskip 5pt
\item{} 800$^{o}$ F = \underbar{\hskip 50pt} $^{o}$ R
\end{itemize}

\underbar{file i00639}
%(END_QUESTION)





%(BEGIN_ANSWER)

\begin{itemize}
\item{} 450$^{o}$ F = \underbar{\bf 232.22}$^{o}$ C
\vskip 5pt
\item{} 2135 K = \underbar{\bf 1861.85}$^{o}$ C
\vskip 5pt
\item{} 800$^{o}$ F = \underbar{\bf 1259.67}$^{o}$ R
\end{itemize}

2 points for C-F conversion, 4 points each for K-C and F-R conversions.
