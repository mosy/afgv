Solve for the value of $x$ in this equation:

$${e^x - e^{-x} \over 3} = -1$$

Note: this is a rather challenging problem, requiring you to use (among other things) the quadratic formula in your solution!

\underbar{file i01323}
%(END_QUESTION)





%(BEGIN_ANSWER)

$$x = {\sqrt{13} - 3 \over 2}$$
