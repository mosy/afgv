An analog-to-digital converter has a 16-bit binary output and an analog input voltage range of 0.0 to + 10.0 volts.  Calculate:

\begin{itemize}
\item{} The digital output (in hexadecimal) at 0.0 volts analog input = \underbar{\hskip 50pt}
\vskip 10pt
\item{} The digital output (in hexadecimal) at 10.0 volts analog input = \underbar{\hskip 50pt}
\vskip 10pt
\item{} The digital output (in hexadecimal) at 3.4 volts analog input = \underbar{\hskip 50pt}
\vskip 10pt
\item{} The analog input corresponding to a digital output of {\tt 3D6A} = \underbar{\hskip 50pt} volts
\vskip 10pt
\item{} The analog input corresponding to a digital output of {\tt C005} = \underbar{\hskip 50pt} volts
\end{itemize}

\underbar{file i02661}
%(END_QUESTION)





%(BEGIN_ANSWER)

\begin{itemize}
\item{} The digital output (in hexadecimal) at 0.0 volts analog input = \underbar{\bf 0000}
\vskip 10pt
\item{} The digital output (in hexadecimal) at 10.0 volts analog input = \underbar{\bf FFFF}
\vskip 10pt
\item{} The digital output (in hexadecimal) at 3.4 volts analog input = \underbar{\bf 5709} or \underbar{\bf 570A}
\vskip 10pt
\item{} The analog input corresponding to a digital output of {\tt 3D6A} = \underbar{\bf 2.399} volts
\vskip 10pt
\item{} The analog input corresponding to a digital output of {\tt C005} = \underbar{\bf 7.501} volts
\end{itemize}
