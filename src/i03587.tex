An analog-to-digital converter has a 12-bit binary output and an analog input voltage range of 0.0 to + 5.0 volts.  Calculate:

\begin{itemize}
\item{} The digital output (in hexadecimal) at 0.0 volts analog input = \underbar{\hskip 50pt}
\vskip 10pt
\item{} The digital output (in hexadecimal) at 5.0 volts analog input = \underbar{\hskip 50pt}
\vskip 10pt
\item{} The digital output (in hexadecimal) at 2.1 volts analog input = \underbar{\hskip 50pt}
\vskip 10pt
\item{} The analog input corresponding to a digital output of {\tt 2D0} = \underbar{\hskip 50pt} volts
\vskip 10pt
\item{} The analog input corresponding to a digital output of {\tt F14} = \underbar{\hskip 50pt} volts
\end{itemize}

\underbar{file i03587}
%(END_QUESTION)





%(BEGIN_ANSWER)

\begin{itemize}
\item{} The digital output (in hexadecimal) at 0.0 volts analog input = \underbar{\bf 000}
\vskip 10pt
\item{} The digital output (in hexadecimal) at 5.0 volts analog input = \underbar{\bf FFF}
\vskip 10pt
\item{} The digital output (in hexadecimal) at 2.1 volts analog input = \underbar{\bf 6B7} or \underbar{\bf 6B8}
\vskip 10pt
\item{} The analog input corresponding to a digital output of {\tt 2D0} = \underbar{\bf 0.8791} volts
\vskip 10pt
\item{} The analog input corresponding to a digital output of {\tt F14} = \underbar{\bf 4.713} volts
\end{itemize}

%(END_ANSWER)
