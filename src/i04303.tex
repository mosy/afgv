$$V = {{2.303 R T} \over {nF}} \left(7 - \hbox{pH}_1 \right)$$

$$V = {{(2.303) (8.315) T} \over {(1)(96485)}} \left(7 - \hbox{pH}_1 \right)$$

\begin{itemize}
\item{} pH = 5.82 ; $T$ = 25 $^{o}$C ; Nernst voltage = \underbar{\bf 69.83} mV
\vskip 5pt
\item{} pH = 8.75 ; $T$ = 43 $^{o}$C ; Nernst voltage = \underbar{\bf $-$109.8} mV
\vskip 5pt
\item{} pH = 11.23 ; $T$ = 95 $^{o}$F ; Nernst voltage = \underbar{\bf $-$258.7} mV
\vskip 5pt
\item{} pH = 3.29 ; $T$ = 122 $^{o}$F ; Nernst voltage = \underbar{\bf 237.9} mV
\end{itemize}

\vskip 10pt

\filbreak

Manipulating the Nernst equation to solve for pH:

$$V = {{(2.303) (8.315) T} \over {(1)(96485)}} \left(7 - \hbox{pH}_1 \right)$$

$${{96485 V} \over {(2.303) (8.315) T}} = \left(7 - \hbox{pH}_1 \right)$$

$$\hbox{pH} = 7 - {{96485 V} \over {(2.303) (8.315) T}}$$

\begin{itemize}
\item{} Nernst voltage = 105 mV ; $T$ = 25 $^{o}$C ; Solution = \underbar{\bf 5.23} pH
\vskip 5pt
\item{} Nernst voltage = 233 mV ; $T$ = 12 $^{o}$C ; Solution = \underbar{\bf 2.88} pH
\vskip 5pt
\item{} Nernst voltage = $-40$ mV ; $T$ = 77 $^{o}$F ; Solution = \underbar{\bf 7.68} pH
\vskip 5pt
\item{} Nernst voltage = $-185$ mV ; $T$ = 131 $^{o}$F ; Solution = \underbar{\bf 9.84} pH
\end{itemize}
