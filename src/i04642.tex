Write the equation for a direct-acting proportional controller, containing the following variables: PV, SP, $K_p$ (gain), $m$ (output), and $b$ (bias):

\vskip 100pt

\underbar{file i04642}
%(END_QUESTION)





%(BEGIN_ANSWER)

$m = K_p(\hbox{PV} - \hbox{SP}) + b$

%(END_ANSWER)
